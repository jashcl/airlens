# Dataset Notes

Dataset selected:
Air Quality in India 2015-2024

Primary file:
city_day.csv

This file is enough for the complete MVP.

Expected columns:
- City
- Date
- PM2.5
- PM10
- NO
- NO2
- NOx
- NH3
- CO
- SO2
- O3
- AQI
- AQI_Bucket

Use city_day.csv for:
- AQI trend analysis
- pollutant comparison
- PM2.5 and PM10 analysis
- AQI bucket distribution
- worst pollution days
- anomaly detection
- research brief generation
- blog/article drafting
- stakeholder outreach content
- Markdown/LaTeX report generation

Optional future file:
city_hour.csv

Use city_hour.csv later for:
- hourly pollution patterns
- morning vs evening comparison
- peak pollution hours

Do not start with city_hour.csv.
Build the complete project using city_day.csv first.
