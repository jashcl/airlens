# AirLens Project Plan

## Goal

Build a polished full-stack climate-tech research and outreach platform using real Indian air quality data.

## Fixed dataset

Primary dataset file:
city_day.csv from Air Quality in India 2015-2024.

## Build phases

### Phase 1
Backend scaffold and frontend scaffold.

### Phase 2
Dataset upload, cleaning, validation, and data quality summary.

### Phase 3
Analysis APIs and dashboard visualizations.

### Phase 4
Research brief generation.

### Phase 5
Content studio for blogs, public notes, LinkedIn posts, and stakeholder notes.

### Phase 6
Outreach CRM for stakeholder records and email draft generation.

### Phase 7
Research log for research activities, sources, notes, tags, and draft status.

### Phase 8
Biofiltration/intervention comparison module.

### Phase 9
Markdown and LaTeX report export.

### Phase 10
Deployment, README, screenshots, interview explanation, and resume bullets.
